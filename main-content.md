# Тема: JavaScript: основи клієнтського програмування

## 1. Рівень вивчення
Дана тема відноситься до **базового рівня**, оскільки входить до переліку тем, рекомендованих програмою МОН України для учнів 10–11 класів. Вона є складовою розділу «Веб-технології» та спрямована на формування базових знань і практичних навичок створення інтерактивних веб-сторінок.

---

## 2. Мета навчання
Метою вивчення теми є формування в учнів уявлення про можливості мови програмування JavaScript, розвиток алгоритмічного мислення, а також набуття практичних навичок створення інтерактивних елементів веб-сторінок.

---

## 3. Очікувані результати навчання
Після опрацювання теми учні:

- розуміють призначення мови JavaScript та її роль у веб-розробці;
- розрізняють HTML, CSS і JavaScript;
- вміють створювати прості програми мовою JavaScript;
- використовують змінні, типи даних, умовні оператори та цикли;
- створюють та використовують функції;
- працюють із подіями користувача;
- змінюють елементи веб-сторінки за допомогою DOM;
- створюють прості інтерактивні веб-проєкти.

---

## 4. Детальний план теми

### 4.1. Вступ до JavaScript
**Ключові поняття:** JavaScript, браузер, скрипт  
**Практика:** створення першої програми (`alert()`)

### 4.2. Змінні та типи даних
**Ключові поняття:** змінні, number, string, boolean  
**Практика:** створення змінних, обчислення

### 4.3. Умовні оператори
**Ключові поняття:** if, else  
**Практика:** перевірка умов

### 4.4. Цикли
**Ключові поняття:** for, while  
**Практика:** повторення дій, таблиця множення

### 4.5. Функції
**Ключові поняття:** функція, параметри  
**Практика:** створення власних функцій

### 4.6. Події
**Ключові поняття:** click, події  
**Практика:** кнопка з дією

### 4.7. DOM
**Ключові поняття:** елементи сторінки  
**Практика:** зміна тексту, стилю

### 4.8. Міні-проєкт
**Практика:** створення інтерактивної сторінки (калькулятор, тест, кнопка)
